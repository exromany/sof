# Сервис просмотра stack overflow

## Запуск

Для запуска сервиса нужен `nodejs`

В папке проекта нужно выполнить:

```bash
npm install

npm start
```

Сервис запустится по адресу [http://localhost:3000](http://localhost:3000)

## Запуск сервиса из скомпилированных файлов

```bash
cd dist/
python -m SimpleHTTPServer 3000
```
